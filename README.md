# AutoML_ITMO

Описание задания
Основная задача - выбрать и решить соревнование с платформы Kaggle.com, используя два подхода:
1. Подготовить базовое решение (бейзлайн) с помощью Light Auto ML (LAMA)
2. Реализовать альтернативное решение без использования LAMA


Ссылка на датасет https://www.kaggle.com/competitions/playground-series-s4e12/overview
